Breve descrição do que foi feito:

Foi feita uma página web, com informações diversas sobre a obra, como o autor, informações gerais, personagens e localidade e com um soneto sobre a obra e seus personagens. Além disso, foi feita uma área de comentários para a interação de leitores e complemento deles.


Etapas desenvolvidas para o planejamento e produção:

  1. Planejar como seria feito o site.
  2. Criação do site.
  3. Criação da API para comentários.
  4. Implementação da API no site.
  5. Organizar os conteúdos a serem discutidos no site.
  6. Escrita do soneto.
  7. Escrever os textos para o site.
  8. Alocação da API e do site na web.
  9. Apresentação do site.

Recursos materiais e digitais utilizados:

Editores de texto para programar. Serviços de alocação para a API e para o site. Materiais de escrita para a escrita do soneto. Computadores e navegadores para implementar tudo.


Resultado e contribuição da produção:

Apresentar informações sobre a obra, interação entre leitores com comentários e compartilhamento de um soneto relacionado com a história.


Principais aprendizagens:

Prática de criar sites e APIs, a integrando no anterior, de discorrer sobre características de uma obra a partir de sua leitura anterior e de escrever textos literários sobre a história e personagens da obra.
